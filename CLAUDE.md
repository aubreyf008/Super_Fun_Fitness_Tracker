This is a personal desktop fitness tracker app built with Electron.
User starting stats: 180 lbs, 5'9", 18-19% body fat, cutting to ~12-13%
Daily targets: 2,300 calories, 190g protein, 215g carbs, 68g fat
Stack: Electron, HTML/CSS/JS, electron-store for local data, Anthropic API for AI features
UI: Dark athletic theme, bold, high contrast
Always ask before making major architectural decisions.
